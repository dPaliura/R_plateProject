Описание задачи ПО

Программное обеспечение решает задачу о распределении температур на плоской, изолированной пластине,
соответственно задача рассматривается на плоскости. Стороны пластины находятся под воздействием температур.
Задача состоит в нахождении температур после момента достижения термодинамического равновесия во внутренних
точках пластины, которые находятся в узлах сетки, которая имеет постоянные шаги по горизонтали и по вертикали.


Описание модели решения задачи

Задача решается методом сетей, ищется приближенное решение уравнения Лапласа на плоскости в прямоугольной области.
Для достижения точности вычислений используется метод Гаусса-Зейделя. 


Замечание 1 (про точность решения)

Отметим, что заданная точность является только точностью вычислений, но не точным решением самой задачи, поскольку
для достижения необходимой точности значений температур в указанных точках необходимо увеличивать обьем узлов сетки.
Более того, без проведения дополнительного вычисления с большим числом узлов сетки невозможно определить погрешность
отосительно точного решения задачи. Для нахождения достаточно точных значений применяется принцип Рунге.
Данная возможность присутствует в данном ПО, но является недоступной без внесения изменений в программный код.
Эта опция не доступна в силу того, что ее реализация является довольно сомнительной и очень емкой в контексте
вычислений (при некоторых условиях может вызвать зависание компьютера в силу отсутствия оперативной памяти на 
создание матриц очень большого размера).

Инструкции по установке программы
1) Выбрать директорию размещения проекта на компьютере.
2) Загрузить все файлы проекта в заданную директорию.
3) Открыть файл Main.R в произвольном текстовом редакторе (редакторе кода)
4) Заменить параметр команды setwd (set working directory) (23 строка) на строку с директорией, куда было загружено данный проект.
При этом каждый обратный слэш ("\") в строке с заданной вами директорией должен быть заменен на двойной 
обратный слэш ("\\").
5) Сохранить изменения.
Для проверки правильности установки достаточно выполнить измененную команду, если не появилась ошибка, то установка 
произошла правильно.


Инструкция по использованию программы

Для выполнения программы используется указанный выше файл Main.R
RStudio является оболочкой программы R, позволяет работать с программой в удобном графическом интерфейсе, редактировать и выполнять
отдельные файлы данного языка (расширение .R) и множество других возможностей. Здесь не описано работу RStudio. Но в дальнейшем
будет предполагаться, что работа с ПО происходит в RSudio. Отметим различие в выполнении программы в R и RStudio: 
для выполнения в R необходимо вручную вводить каждую команду файла Main.R, когда в RStudio достаточно открыть этот файл и выполнять
каждую команду при помощи средств среды выполнения.
Для выполнения команды нужно установить указатель на строке, в которой находится команда, и нажать на кнопку "Run", на которой
изображена зеленая стрелочка, или же комбинацией клавиш Ctrl + Enter.

Каждая команда подписана комментарием, таким образом нужно выполнить 3 первых команды:
1) установить рабочую директорию;
2) импортировать весь необходимый код программы source("Controller.R", echo = FALSE);
3) запустить выполнение программы выполнением команды Controller.start().

Для изменения параметров решаемой задачи необходимо редактировать файл Data.R, который содержит данные о решаемой задаче.
Все значения создаваемых переменных в данном файле соответствуют параментрам задачи, для их изменения достаточно заменить их
числовые значения на необходимые.
Описание переменных содержатся в их названиях.
После изменения каки-либо значений необходимо сохранить изменения в данном файле.
После изменения файла нет нужды в перезапуске чего-либо, значения параметров берутся из этого файла уже в процессе решения задачи.

В консоли выводится текст с некоторыми данными о решаемой задачи, включая входные данные и результат решения.
Для изменения языка программы нужно выполнить дополнительную команду set_lang, в которой указан параметр
соответствующий локализации. Для каждой реализованной интернационализации уже написана команда set_lang в коде файла Main.R и
находится после команды Controller.start().
Каждая команда главого файла закомментирована ее кратким описанием.
По умолчанию задано Англоязычную локализацию.
Изменение языка выполняется прежде выполнения команды Controller.start().
